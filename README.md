# NoBrain Meals

NoBrain Meals is a lightweight landing-page project for a low-cost digital food-planning product.
It is built to be uploaded to GitHub and deployed on a free tier such as Vercel or GitHub Pages.

## What it includes
- Static landing page
- Browser-native audio transcript playback
- Product content files
- Positioning and roadmap notes
- Tiny local Node server for testing

## Run locally

### Option 1: Node server
```bash
node server.js
```
Then open `http://localhost:3000`

### Option 2: Static preview
Open `index.html` in a browser.

## Deploy
- Upload all files to a GitHub repo
- Import repo into Vercel
- No API keys required

## Replace before launch
- Update contact email / CTA link in `index.html`
- Adjust pricing and copy if needed
